## Data (scratch folder, for J it is /scratch/general/vast/u6025801/)

- `data/`
    - `investors/`: Processed institutional trading data and regression outputs.
        - `inst_monthly/`: Monthly aggregated institutional trades (generated by python pipeline).
        - `lasso_out_monthly/`: Lasso regression outputs/residuals for institutional trades, saved yearly.
    - `jkp/`: JKP (Jensen, Kelly, Pedersen) factor data (contains factor returns, cluster maps, and sign files).
    - `prices/`: Stock price and return data.
        - `daily_byungwook/`: Custom daily price and return datasets.
            - `daily_return/`
            - `stock_day_prices/`
        - `daily_compustat/`: Daily price data from Compustat. Saved in monthly partitions (`yyyymm=200101` to `yyyymm=202412`).
        - `raw/`: Unprocessed/raw price data.
    - `raw/`: Raw, original source files for trades.
        - `parsed_trades/`: Initial parsed trade datasets, split by market.
            - `ksq/`: KOSDAQ
            - `stk/`: KOSPI
        - `raw_data/`: The original, untouched raw trade data files.
            - `ksq/`: KOSDAQ
            - `stk/`: KOSPI
